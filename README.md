# Avance Proyecto Final 

Dentro de esta rama se detalla el avance del proyecto final para Programación Orientada a Objetos **(POO)**

## Análisis de Requerimientos 
>boceto
Este sistema tiene como objetivo gestionar préstamos de videojuegos, que permita registrar y consultar videojuegos, así como también gestionar préstamos realizados


---

## Diseño de Clases y Objetos

>Boceto
Para este sistema existen 5 clases:

>- Videojuego
>- Cliente
>- Empleado 
>- Prestamo 
>- Inventario
